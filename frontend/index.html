Metodologi Penelitian

Penelitian ini menerapkan pendekatan machine learning berbasis teks dengan algoritma Support Vector Machine (SVM) untuk melakukan klasifikasi topik dan sentimen komentar publik terkait bencana banjir bandang di wilayah Sumatra.

1. Pengumpulan Data

Data diperoleh melalui proses web scraping menggunakan YouTube Data API v3 pada kanal berita yang membahas peristiwa banjir bandang di Sumatra.

  • Sumber data berupa komentar publik
  • Total data lebih dari 14.000 komentar
  • Data mencakup komentar utama dan balasan
youtube.commentThreads().list(
  videoId=video_id,
  part="snippet,replies",
  maxResults=100
)

2. Preprocessing Data Teks

Data mentah dibersihkan secara sistematis untuk mengurangi noise dan meningkatkan kualitas fitur teks sebelum proses pemodelan.

  • Case folding (huruf kecil)
  • Penghapusan URL, emoji, angka, dan simbol
  • Normalisasi slang menggunakan kamus tidak baku
  • Tokenisasi dan stopword removal (negasi dipertahankan)
  • Stemming Bahasa Indonesia (Sastrawi)
  • Filtering komentar pendek (< 5 kata)
def preprocess(text):
  text = text.lower()
  text = remove_url(text)
  text = normalize_slang(text)
  text = stem(text)
  return text

3. Pelabelan Data

Proses pelabelan dilakukan secara semi-otomatis untuk efisiensi dan konsistensi data latih.

  • Pelabelan topik menggunakan Sentence-BERT (cosine similarity)
  • Empat kategori topik utama yaitu: Simpati & Doa, Dampak & Kondisi Lapangan, Kerusakan Lingkungan, dan Kritik Pemerintah
  • Pelabelan sentimen menggunakan IndoBERT
  • Refinement khusus kategori Simpati & Doa
  • Validasi manual + Cohen‚Äôs Kappa
label = argmax(
  cosine_similarity(
    comment_embedding,
    label_embedding
  )
)

Evaluasi performa model dilakukan untuk mengukur kemampuan generalisasi terhadap data yang belum pernah dilihat sebelumnya.

  • Confusion Matrix
  • Precision, Recall, F1-score
  • Accuracy & Macro F1
  • 5-Fold Cross Validation
